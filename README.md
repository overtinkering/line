# line
A tiny command-line music sequencer.

+ Sends MIDI messages to channel 1.

+ 60 bpm by default

+ 1 instrument


### Live coding

Type in command-line.

**To send a MIDI message:**

`45 46 47 48 // midi values as 4 1/4 notes`

`45 .34 35. 48 49 // 1/4 1/8 1/8 1/4 1/4 notes`

**To set bpm:**  

`b120 // 120 bpm`

**To exit:**

`e`  

**To display options:**

`o`
